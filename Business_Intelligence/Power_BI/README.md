# Project Overview
Built an end-to-end SQL + Power BI analytics project to identify profit leakage, discount risk, and shipping inefficiencies in retail operations. The project is designed to solve a real business problem: helping stakeholders move beyond sales growth and focus on where profit is being lost, why it is happening, and what actions should be prioritized first.

## Business Impact
This project highlights how a business can improve profitability by pinpointing loss-making regions, sub-categories, discount patterns, and shipping decisions. Instead of just reporting metrics, the dashboard supports decision-making through root-cause analysis and action-oriented insights.

## Technical Scope
- Used SQL for data import, validation, and preparation before analysis
- Resolved data loading issues during CSV import by identifying a delimiter mismatch, validating datatypes, and ensuring records were imported correctly into SQL tables
- Built interactive Power BI dashboards using Power Query, data modeling, and DAX measures
- Developed KPI tracking for Sales, Profit, Profit Margin, Discount, Loss Orders, Loss Amount, Shipping Cost, and Delivery SLA metrics
- Created business-focused visuals including executive scorecards, Pareto analysis, diagnostic matrices, decomposition tree, scatter analysis, and action recommendation views

## What this project demonstrates
- Strong business framing, not just dashboard building
- Practical use of SQL + Power BI for analytics workflows
- Ability to connect data cleaning, transformation, KPI design, and storytelling into one decision-support solution
- Recruiter-relevant skills: SQL, Power BI, Power Query, DAX, KPI reporting, root-cause analysis, and business recommendations
